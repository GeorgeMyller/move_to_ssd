Move_To_SSD

Python script to migrate folders and apps from macOS to an external SSD with automatic symlinks.

Free Up Space. Gain Speed.

Automate the transfer of files, folders, and apps from your Mac to an external SSD safely and transparently.

View on GitHub

Features

Smart Transfer

Move user folders, apps, caches, and other large files to the external SSD.

Automatic Symlinks

Creates symlinks at the original locations, keeping everything transparent for the system and apps.

Detailed Report

Shows a summary of successes, failures, and skipped items at the end.

Customizable Setup

Choose which folders, apps, and files to migrate by editing the script.

How It Works

1. Configure

Edit ssd_transfer.py to set your SSD name and the folders/apps to migrate.

2. Run

Run the script in Terminal:
python3 ssd_transfer.py

3. Confirm

Check the summary and confirm to start the migration.

Recommendations

  • Back up important data before migrating.
  • Close apps that may be using files to be moved.
  • To move apps, run with sudo.
