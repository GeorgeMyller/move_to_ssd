Move_To_SSD

Script Python para migrar pastas e aplicativos do macOS para SSD externo com symlinks automáticos.

Libere Espaço. Ganhe Velocidade.

Automatize a transferência de arquivos, pastas e aplicativos do seu Mac para um SSD externo de forma segura e transparente.

Ver no GitHub

Funcionalidades

Transferência Inteligente

Move pastas de usuário, aplicativos, caches e outros arquivos grandes para o SSD externo.

Symlinks Automáticos

Cria links simbólicos nos locais originais, mantendo a transparência para o sistema e aplicativos.

Relatório Detalhado

Exibe um resumo de sucesso, falhas e itens ignorados ao final do processo.

Configuração Personalizável

Escolha quais pastas, apps e arquivos migrar editando o script.

Como Usar

1. Configure

Edite o arquivo ssd_transfer.py para definir o nome do SSD e as pastas/apps a migrar.

2. Execute

Rode o script no Terminal:
python3 ssd_transfer.py

3. Confirme

Confira o resumo das operações e confirme para iniciar a migração.

Recomendações

  • Faça backup dos dados importantes antes de migrar.
  • Feche aplicativos que possam estar usando os arquivos a serem movidos.
  • Para mover apps, execute com sudo.
